"O Canhão é uma arma de longo alcance com ataques poderosos no Brawlhalla, capaz de controlar o espaço e causar grandes danos aos inimigos."

O Canhão é um morteiro pesado. Seu grande tamanho é um tanto enganador, pois os carregadores de canhões atacam com explosões rápidas ou até mesmo golpeiam os oponentes com o peso da própria arma.

Os ataques básicos do Canhão são capazes de derrubar inimigos com facilidade, e os ataques especiais podem causar danos devastadores a uma grande área do campo de batalha.

Kits

Neutral light

O usuário entrega uma explosão na frente deles. Se um inimigo é pego no ataque, o usuário salta para a frente para entregar um segundo ataque, aplicando knockback e movendo grandemente o inimigo para longe da posição inicial.

Side light

O usuário salta para a frente enquanto balança o canhão para cima e diagonalmente em relação ao usuário.

Down light

O usuário pula para frente e acerta o canhão no chão, derrubando inimigos para cima e diagonalmente em relação ao usuário.

Neutral Air

O usuário dispara uma pequena explosão acima deles. O usuário também é empurrado para baixo via recuo.

Side Air

O usuário usa a explosão do canhão para se impulsionar para a frente, atingindo qualquer um no caminho. A explosão do canhão não causa danos.

Down Air

O usuário dispara uma pequena explosão diagonalmente abaixo deles.

Recovey

O usuário dispara uma pequena explosão diagonalmente abaixo deles.

Ground Pound

O usuário cai com o canhão abaixo deles. Se o ataque se conectar, uma explosão enviará o inimigo para baixo. Este ataque criará uma pequena explosão se atingir o solo.
