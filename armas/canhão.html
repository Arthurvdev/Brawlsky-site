Nomenclaturas

"O Canhão é uma arma de longo alcance com ataques poderosos no Brawlhalla, capaz de controlar o espaço e causar grandes danos aos inimigos."

O Canhão é um morteiro pesado. Seu grande tamanho é um tanto enganador, pois os carregadores de canhões atacam com explosões rápidas ou até mesmo golpeiam os oponentes com o peso da própria arma.

Os ataques básicos do Canhão são capazes de derrubar inimigos com facilidade, e os ataques especiais podem causar danos devastadores a uma grande área do campo de batalha.

Kits

Neutral light

O usuário atira com o canhão para frente. Se um inimigo é pego no ataque, o usuário salta para a frente para entregar um segundo ataque (diagonal).

Side light

O usuário salta para a frente enquanto gira o canhão para cima, diagonalmente em relação ao usuário.

Down light

O usuário pula para frente e acerta o canhão no chão, jogando o oponente para cima.

Neutral Air

O usuário dispara uma pequena explosão acima dele mesmo.

Side Air

O usuário usa a explosão do canhão para se impulsionar para a frente, atingindo qualquer um no caminho.

Down Air

O usuário dispara uma pequena explosão diagonalmente abaixo dele mesmo.

Recovey

O usuário usa a explosão do canhão para atacar diagonalmente no ar.

Ground Pound

O usuário cai com o canhão e usa a sua explosão para enviar o inimigo para baixo. No solo: O ataque criará uma pequena explosão horizontal.
