"Um conjunto de duas botas pesadas com intrincados combos de chute e ataques de entrada ativos."

Como o nome indica, as Botas de Batalha são um conjunto de botas pesadas que se concentram em ataques chamativos e baseados em chutes. Muitos dos ataques leves das Botas de Batalha podem ser alterados com entradas ativas. Esses ataques de entrada ativa são usados principalmente para mudar o posicionamento do usuário após um ataque, dando ao jogador mais aberturas para diferentes combos de chute.

As Botas de Batalha são a primeira arma em Brawlhalla que não são seguradas nas mãos do jogador e, portanto, têm animações de movimento muito únicas para garantir que os usuários de Botas de Batalha não sejam confundidos com jogadores desarmados.

Kits

Neutral light

O usuário solta uma enxurrada rápida de chutes com um pé. Se ele se conectar, o usuário segue com um chute um pouco mais forte do outro pé.

Side light

O usuário avança, girando botas primeiro com dois chutes. Este ataque é afetado pela entrada ativa no hit: ● Input para a frente: O adversário é chutado três vezes, sendo o chute final um salto que torna o usuário no ar. ● Sem input: O adversário é chutado três vezes, o chute final mantendo o usuário aterrado.

Down light

O usuário realiza uma varredura de perna baixa. Este ataque é afetado pela entrada ativa no hit: ● Input para baixo: Após a varredura da perna, o usuário executa um chute para baixo saltando para baixo, fazendo com que o usuário termine aterrado. ● Sem Input: Após a varredura da perna, o usuário executa um chute lateral de salto, fazendo com que o usuário termine no ar.

Neutral Air

O usuário executa um chute de backflip sobre a cabeça. Isso atinge na frente, acima e atrás do usuário.

Side Air

O usuário executa dois chutes para frente, um com cada pé. Este ataque é afetado pela entrada ativa no hit: ● Input para a frente: Após os dois primeiros chutes, o usuário ataca com um chute giratório, fazendo-os viajar para a frente. ● Sem Input: Após o primeiro a chutar, o usuário executa um chute mais forte com os dois pés. Isso derruba o oponente e mantém o usuário no mesmo lugar.

Down Air

O usuário chuta com dois chutes diagonais para baixo. Este ataque é afetado pela entrada ativa no hit: ● Input para baixo: O usuário continuará diagonalmente para baixo através do adversário com o segundo chute. ● Sem Input: O usuário saltará do adversário com o segundo chute.

Recovey

O usuário executa três chutes giratórios para cima.

Ground Pound

O usuário mergulha para baixo, realizando uma queda de perna quando o ataque é liberado - essa queda de perna tem um alcance de ataque maior se usada antes do solo. O usuário também é capaz de se mover para a esquerda e para a direita enquanto segura o ataque. Movimento

● Correr para a frente fará o jogador correr com as mãos para fora atrás das costas. ● Correr para trás faz com que o jogador faça um backflip.
